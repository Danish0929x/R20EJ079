# Server Side

## Installation

To install the Node modules, use the following command:

```bash
npm install
```


To run the server:

```bash
nodemon server.js
```

# Client Side

## Installation

To install the Node modules, use the following command:

```bash
npm install
```


To run the development server:

```bash
npm start
```
